Berikut ini adalah beberapa perbedaan antara HTML dan CSS

HTML

  • Memiliki ekstensi .HTML
  • HTML memerlukan tag deklarasi di setiap filenya
  • Bersifat tidak Case Sensitive
  • Digunakan untuk membangun konten pada website

CSS

  • Memiliki ekstensi .CSS
  • CSS tidak memerlukan deklasari
  • bersifat Case Sensitive
  • Digunakan untuk menemtukan bagimana elemen HTML akan ditampilkan
